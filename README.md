# HexSoftwares_Analyzing_and_Visualizing_Movie_Ratings

## Overview
This project performs an analysis on an IMDb Movie Ratings dataset and creates a visualization using Python. The goal is to explore the dataset, clean the data, and visualize ratings distributions and identify top-rated movies and genres.

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
The dataset is taken from Kaggle and loaded directly from the GitHub repository
